打开Excel文件后，在`运算`选项卡下的筛选式输入框按照准则输入筛选式
## 原理
使用python的`eval()`函数对式子进行计算

详见 [excel.py](https://github.com/Loser123zbx/ZjangDataViewer/blob/master/src/excel.py)

# 自定义准则填写

`x` 表示项内所有值

## 支持的符号

### 逻辑运算

等于 `=`  	

大于等于 `>=`	

小于等于 `<=`

大于 `>`

小于 `<`

不等于 `!=`

### 数学运算

 加 `+`

 减 `-`

 乘 `*`

 除 `/`

 次幂 `**`

 整除 `//`	

 取余 `%`  

### 逻辑运算:(前后要空格)

 和  `and`
 
 或 `or`

 非 `not`

### 语法

[筛选部分] # <运算部分>
筛选部分必填，其他选填，如：

输出所选列中大于114的值与其对应的名
> x > 114

输出所选列中大于114的值与其对应的名，并且输出与114的差
> x > 114 # x - 114

输出所选列中大于114而且小于514的值与其对应的名
> x > 114 and x < 514
